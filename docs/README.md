# Build Documentation

This subdirectory is used to build the documentation.

## Dependencies

- [doxide](https://github.com/lawmurray/doxide)
- [mkdocs](https://github.com/mkdocs/mkdocs)

To build the documentation, run:

```bash
doxide build
mkdocs build
```

To serve the documentation locally, run:

```bash
mkdocs serve
```
